# Firebase Data Models

Ce dossier contient les modèles de données pour Firebase, organisés en sous-dossiers selon les différentes collections de la base de données.

## Structure

```
models/firebase/
├── users/
├── places/
├── categories/
├── tours/
├── events/
├── comments/
├── ratings/
├── favorites/
├── interests/
└── notifications/
```
